# Unit 3 Checklist: Skills API

Use this at the end of the unit to verify you're ready to move on.

- [ ] `skills-api` project created and compiles without errors
- [ ] `/api/system/info` endpoint responds with system data
- [ ] `/api/system/disk` endpoint responds with disk usage
- [ ] `/api/services/status` validates against allowlist, returns `403` if blocked
- [ ] `/api/logs/tail` validates source against allowlist, returns `403` if blocked
- [ ] Audit log written for every request (success and failure)
- [ ] Docker image builds without errors
- [ ] Container running on lab server or VM
- [ ] API reachable from laptop via LAN/local IP
- [ ] Integration with Unit 2 client tested end-to-end
